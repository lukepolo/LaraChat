Data Binding

  • data
  • methods
  • computed

To bind some data to the template, all you have to do is: 



  {{ message }}



var app = new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})

Now, when you change message inside of your data, the message will automatically update!
